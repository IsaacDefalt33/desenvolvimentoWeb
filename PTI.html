Explicação da Atividade 1 - Flexbox e Media Queries

Objetivo: Criar uma loja de card games responsiva usando Flexbox e media queries.

Conceitos aplicados:

  • Flexbox: Utilizado para organizar os cards em linhas responsivas
  • Media Queries: Ajustam o layout para diferentes tamanhos de tela
  • Propriedade flex: Controla o crescimento, encolhimento e tamanho base dos elementos
  • Variáveis CSS: Usadas para manter consistência nas cores

Breakpoints implementados:

  • > 1200px: 4 cards por linha, fundo padrão
  • 768px - 1200px: 3 cards por linha, fundo roxo (ametista)
  • 576px - 768px: 2 cards por linha, fundo amarelo (girassol)
  • < 576px: 1 card por linha, fundo laranja (cenoura), menu sanduíche
/* Cálculo do tamanho dos cards */
.card-container {
  flex: 0 0 calc(25% - 15px);
}

/* Explicação:
  - 0: não cresce
  - 0: não encolhe
  - calc(25% - 15px): largura base de 25% menos 15px
    para compensar o gap de 20px (10px de cada lado) */

Piet Mondrian - Composição Abstrata

Uma interpretação moderna do estilo De Stijl usando CSS Grid

Piet Mondrian (1872 – 1944)

Pintor holandês que se tornou uma das figuras mais influentes da arte abstrata no século XX. Fundador do movimento De Stijl ("O Estilo"), que defendia a arte baseada em formas geométricas puras e nas cores primárias (vermelho, azul e amarelo), combinadas com preto, branco e cinza.

Suas obras mais conhecidas são compostas por retângulos delimitados por linhas pretas e preenchidos com cores primárias ou áreas brancas. Ele via essa redução como uma forma de expressar harmonia universal, quase como se fosse uma linguagem visual matemática.

Conceitos de CSS Grid aplicados:

  • grid-template-columns: define o número e tamanho das colunas
  • grid-template-rows: define o número e tamanho das linhas
  • grid-column: define onde uma célula começa e termina horizontalmente
  • grid-row: define onde uma célula começa e termina verticalmente
  • gap: espaçamento entre as células da grade
  • fr units: unidade fracionária que distribui o espaço disponível

Explicação da Atividade 2 - CSS Grid (Arte Mondrian)

Objetivo: Criar uma obra de arte inspirada em Piet Mondrian usando CSS Grid.

Conceitos aplicados:

  • CSS Grid Layout: Sistema bidimensional para criar layouts complexos
  • Grid de 12x12: Grade complexa que permite composições sofisticadas
  • Cores primárias: Vermelho, azul e amarelo conforme o estilo De Stijl
  • Controle preciso: Cada célula é posicionada exatamente com grid-column e grid-row

Código da grade principal:

.mondrian-grid {
  display: grid;
  grid-template-columns: repeat(12, 1fr);
  grid-template-rows: repeat(12, 1fr);
  gap: 8px;
  height: 600px;
}

/* Posicionamento de uma célula específica */
.cell-1 {
  grid-column: 1 / 4; /* da linha 1 à linha 4 */
  grid-row: 1 / 3; /* da linha 1 à linha 3 */
}
